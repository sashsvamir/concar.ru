
# Information


This widget was generated by `arrilot/laravel-widgets` extension, so there two files:
1. class file: `app/Widgets/FeedbackForm.php`
2. view file: `resources/views/widgets/feedback_form.blade.php`





# Installation


To install widget, you should compile `js` and `sass` sources with `laravel mix` and render it in view template.


Adding to `webpack.mix.js`:
```js
mix.js('app/Widgets/FeedbackForm/src/feedback-form.js', 'public/js')
    .sass('app/Widgets/FeedbackForm/src/feedback-form.sass', 'public/css');
```


Adding to view template:
```blade
@widget('feedback-form.widget')
```





### Note:

You can insert compiling `js` and `sass` sources to common `app.js` using `require` method,
so in that case you no needed to modify your `webpack.mix.js`,
just require `js` and `style` files to main bundle:
```js
// main bundle js file content
// ...
require('../../app/Widgets/FeedbackForm/src/feedback-form.js');
require('../../app/Widgets/FeedbackForm/src/feedback-form.sass');
```

also call `@widget` in view template with corresponding argument to prevent double loading `assets`:
```blade
@widget('feedback-form.widget', ['withAssets' => false])
```
